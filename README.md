# FoodieDB
## Description
これは話題のフォトジェニックな料理をインスタグラムではなく、グルメ図鑑として別に記録するアプリケーションです。

## Feature
1. ユーザー登録
2. 自分で食べた食べ物の画像を投稿
3. 食べたものの情報投稿
  
    - 日時
  
    - コメント
  
    - どのくらいお気に入りか？(☆☆☆☆☆の数)
  
    - Google Mapとの連携
  
4. いいね機能
5. 他のSNSにシェア
